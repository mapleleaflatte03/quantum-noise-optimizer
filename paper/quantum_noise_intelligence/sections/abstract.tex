\title{Physically-Bounded Zero-Noise Extrapolation with Information-Theoretic Model Selection for NISQ Error Mitigation}

\author{The Son Nguyen}
\email{nguyensimon186@gmail.com}
\affiliation{Department of Data Science, Van Lang University, Ho Chi Minh City, Vietnam}

\date{\today}

\begin{abstract}
Quantum error mitigation via zero-noise extrapolation (ZNE) suffers from two fundamental failure modes: extrapolation models that violate physical constraints---producing probabilities exceeding unity or negative expectation values---and the absence of principled model selection criteria for the small-sample regime ($n \leq 7$ data points) inherent to ZNE.
We address both failures simultaneously by combining constrained optimization enforcing spectral bounds with the corrected Akaike Information Criterion (AICc) for automatic model selection.
Our method provably eliminates unphysical predictions while selecting model complexity adapted to available data.
Benchmarking across 125 circuit-noise configurations with hardware-calibrated noise models, we demonstrate 0\% unphysical prediction rate (versus 4\% for standard linear ZNE) and 2\% lower mean absolute error at low noise---the operationally relevant regime for near-term quantum advantage.
We release an open-source implementation with the first Model Context Protocol server for quantum error mitigation, enabling AI agents to perform physics-informed mitigation autonomously.
\end{abstract}
